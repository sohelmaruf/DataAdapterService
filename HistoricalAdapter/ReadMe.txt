Installation
------------
Copy HistoricalAdapter.exe to a folder of your choice.  If desired, add
a shortcut to your startup folder so that it runs automatcially
each time you start your computer.  You may also want to set the
run state on the shortcut properties page to minimized.

Requirements
------------

This application requires Microsoft's .Net framework to run.  .Net is
included with Windows XP service pack 1, or you can download it from
Microsoft (see URL below).  .Net requires Windows 98 or later.
http://msdn.microsoft.com/netframework/downloads/framework1_1/
